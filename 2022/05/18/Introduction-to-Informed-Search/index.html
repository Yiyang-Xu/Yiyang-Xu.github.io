Introduction to Informed Search

Uniformed Search vs Informed Search

  1. Uniformed Search (aka blind search)

  2. Informed Search (aka heuristic search)

    • Search guided by heuristic evaluation of how promising a node is.

Uniformed Search

Dijkstra’s Algorithm. It works by visiting vertices in the graph starting with the object’s starting point. Then repeatedly examinges the closest not-yet examinged vertex, adding its vertices to the set of vertices to be examined. As long as none of the edges have a negative cost, Dijkstra’s Algorithm is guaranteed to find a shortest path from the starting point to the goal.

Greedy Search

Strategy:

Expand node in frontier with smallest heuristic value.

Heuristic, some estimate of how far from the goal any vertex is.

Property:

  1. Instead of selecting the vertex closest to the starting point, it selects the vertex closest to the goal.
  2. Not guaranteed to find a shortest path.
  3. Runs much quicker than Dijkstra’s Algorithm because it uses the heuristic function to guide its way towards the goal very quickly.

In this example, the goal (blue point) is to the south of the starting position (pink one), Greedy Search will tend to focus on paths that lead southwards. Yellow represents those nodes with a high heuristic value (high cost to get to the goal) and black represents nodes with a low heuristic value (low cost to get to the goal).

Problem

When there exists obstacles. Dijkstra’s Algorithm is garanteed to find a shortest path but with a lot work to do.

Greedy search does less work but the path it finds is clearly not as good.

The trouble is that Greedy Best-First-Search is “greedy” and tries to move towards the goal even if it’s not the right path. Since it only considers the cost to get to the goal and ignores the cost of the path so far, it keeps going even if the path it’s on has become really long.

The A* Algorithm

A* is like Dijkstra’s Algorithm in that it can be used to find a shortest path. A* is like Greedy Best-First-Search in that it can use a heuristic to guide itself. In the simple case, it is as fast as Greedy Best-First-Search:
